CSS: SELECTORES III

1. El primer elemento a hijo de p


        p > a:first-of-type {
        color: red;
        font-weight: bold;
      }

2. Checkboxs marcados hijos de un form


        form > input[type="checkbox"]:checked {
        font-weight: bold;
      }

3. Selecciona el último p que sea descendiente de article


        article p:last-of-type {
        background: yellow;
      }

4. Seleccionar el segundo hijo de un elemento section


        section :nth-child(2) {
        background: lightblue;  
        }

5. Seleccionar los hijos pares de un elemento ol


          ol :nth-child(even) {
          background: green;
          }

6. Seleccionar el antepenúltimo li de un ul de longitud indefinida


        ul li:nth-last-child(3) {
        background: orange;
        }

7. El segundo a de dentro de un p


        p a:nth-of-type(2) {
        font-size; 50px;
        }

8. Los button que sean hijos único


        button:only-child {
        background: purple;
        color: white;
        }

9. Los span que sean primeros hijos de un p


        p span:first-child {
        color: red;
        font-weight: bold;
        }

10. Penúltimo hijo de body


        body :nth-last-child(2) {
        background: black;
        color: white;
        }
